Bit Heroes Custom Client


Bhconnection

Get Adobe Flash player

Instructions: Reload this page with HTTPS in front of the URL if it is not already. Click to connect to the prod server (web.bitheroesgame) and load the game data. Wait for the data to load, then click any of the XML books to be prompted to save them. To login replace the default values with your username (use0), user_id (use3), and game_auth_token (use4). These can be found on the official client on Kongregate by inspecting the flashvars with the browser. Press send to send the login request to the server. Note: Don't share your game_auth_token as this is your password encrypted for the use of authenticating for Bit Heroes.
